## Installation

SimpleKit is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'SimpleKit', :git => 'https://github.com/amrOmran89/SimpleKit-iOS.git'
```
